# Blueprint - Agent Army
**Creato:** 22 Marzo 2026 | **Owner:** Sem | **Status:** Planning

## Idea di Business

Sistema multi-agente packaged vendibile a aziende medio-grandi.
Un prodotto che porta l'automazione AI nelle aziende senza che debbano costruirsi l'infrastruttura da zero.

## Modello di Business
- **Freemium onboarding:** azienda carica i suoi dati
- **Diagnostica automatica:** AI analizza e produce video diagnostico
- **Proposta pacchetto agenti:** personalizzata per il cliente
- **Marketplace agenti:** fino a 300 agenti specializzati selezionabili
- **Pricing:** abbonamento mensile per agente attivo

## Agenti Pianificati (marketplace)
- SDR Agent - outreach commerciale
- Email Handler - gestione inbox
- Report Agent - reportistica automatica
- Scraping Agent - raccolta dati
- RAG Agent - knowledge base aziendale
- CRM Agent - gestione trattative
- Social Agent - gestione social media
- ... (fino a 300)

## Stack Tecnico
